Reverse Coding

What is it?

Reverse Coding is an event which tests logical thinking and basic programming skills. The idea behind this event is to figure out the "Black Box".

In other words the participants are given an executable program and are asked to do reverse coding, that is to develop the source code. The event will test your basic logical skills along with pattern analysis and recognition

as you play with the code, identify the patterns and decipher it.
